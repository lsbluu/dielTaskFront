
# Boas-vindas ao repositório do projeto Diel Task!

<details>
   <summary><strong> :desktop_computer:	 Requisitos do Projeto</strong></summary><br />
   Páginas obrigatórias:

    - Lista de Tarefas.

             - Edição de tarefa
             - Remoção de tarefa
             - Campo de busca de tarefas
